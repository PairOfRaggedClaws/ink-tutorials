# ink-tutorials
I think the best way to learn a scripting language is hands on, through doing - and most especially through being able to tinker with clear, isolated examples. These files assume you're brave enough to try scripting and are intended to be used *with* the Inky app, available on the inky website: https://www.inklestudios.com/ink/ . 

Install the app, load these files in them, and go through them, comparing the left and right sides. Not everything has been explained! But I think you'll find there's enough information here that you can figure a lot of extras yourself, from here.

There's some great documentation on the site, too, but to my eyes the full writer's manual is more accessible to someone who is familiar with scripting and with reading scripting manuals. Some of the information is scattered (or circular), and doesn't provide quite as easy an on-ramp as I would like.

Once you've got yourself going, I highly recommend going back to that manual for reference, and to learn some ink's advanced features, like lists and stitches.
